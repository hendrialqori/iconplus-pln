## 📦 Tech Stack

- **Vue.js**
- **Typescript**
- **Vite**

## 🧑‍💻 How to Clone and Run

### 1. Clone the Repository

```
git clone git@github.com:hendrialqori/iconplus-pln.git

cd iconplus-pln
```

### 2. Run the Frontend Client)

```
npm install

npm run dev
```

🌐 Visit the app at: http://localhost:5173
